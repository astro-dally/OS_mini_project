Challenges & Learnings

üêõ

Binary Mode is Critical

Opening files in text mode caused corruption. Using O_BINARY flag preserves exact byte values.

üí°

Buffer Size Matters

Small buffers (1KB) caused slow I/O. Increasing to 8KB-64KB improved performance significantly.

üîê

XOR is Self-Inverse

Discovered that XOR cipher uses the SAME function for encryption and decryption!
